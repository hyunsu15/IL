바둑에서는 복기라는 용어가 있습니다.

바둑 대국이 끝난뒤, 해당 대국의 내용을 검토하기 위하여 두었던 순서대로 다시 두어보는 일.

여기서 '검토'만 잘 된다면, 실력을 늘릴수 있는 방법이다.



이걸 프로그래밍에 적용해보면서, 실력을 늘리고,  똑같은 실수를 줄이고, 나아가는 방법를 취해보기 위한 세션입니다.
